## Setup

+ Fork the poject

+ Create a new branch  

+ Clone the project

```bash
  git clone https://github.com/chandel-aman/lost-and-found-pets.git
```

+ Go to the project directory

```bash
  cd frontend
```

+ Install dependencies

```bash
  npm install
```

+ Start the server

```bash
  npm start
```
